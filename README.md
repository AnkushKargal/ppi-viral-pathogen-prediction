# Machine Learning Driven Prediction of Protein-Protein Interactions in Emerging Viral Pathogens

## Overview
This project focuses on predicting protein-protein interactions between viral and human proteins using machine learning techniques.

## Objectives
- Predict virus-host protein interactions
- Perform bioinformatics feature engineering
- Train machine learning models
- Evaluate performance using ROC-AUC and F1-score

## Dataset
Sample protein interaction dataset containing:
- Human protein sequences
- Viral protein sequences
- Interaction labels

## Machine Learning Workflow
1. Data preprocessing
2. Feature extraction
3. Model training
4. Prediction
5. Performance evaluation

## Models Used
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- F1-score
- ROC Curve

## Technologies Used
- Python
- Google Colab
- Pandas
- Scikit-learn
- Matplotlib
